# Spot & Earn

A simple mobile-first web app to track animal sightings and reward kids with small earnings.

## Why?

Spotting certain wild animals when we're driving or out on a bike ride or hike and getting paid small amounts of money for it is something I've been doing for years with my three young kids. It encourages them to look outside (when driving) and is also a fun way to earn more pocket money whilst they're at it!

This app is helpful for me as to keep track of who spotted which animal, and how much I'm due to pay them (usually once per month). It is also a fun (partly) 'agentic' coding experiment, learning as I go.

Feel free to use my idea with your kids (and be careful, as I have run into fields with dozens of deer before, costing you an extra penny or two). This app might be of help as well.

## 🧠 Concept

When the user's children spot animals like deer or hares, they earn money:

* Deer → €1.00
* Hare → €0.50

The app tracks sightings, calculates balances, and keeps a history log.

## 📱 Features

* Quick one-tap logging
* Active child selection
* Automatic balance calculation
* Monthly payout reset
* Persistent history
* Offline-capable (IndexedDB)

## 🧱 Tech Stack

* React + Vite
* TypeScript
* Material UI
* React Router
* IndexedDB (idb)

## 🚀 Getting Started

```bash
npm install
npm run dev
```
