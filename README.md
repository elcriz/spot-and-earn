# Spot & Earn

A simple mobile-first web app to track animal sightings and reward kids with small earnings.

## 🧠 Concept

When children spot animals like deer or hares, they earn money:

* Deer → €1.00
* Hare → €0.50

The app tracks sightings, calculates balances, and keeps a history log.

## 📱 Features

* Quick one-tap logging
* Active child selection
* Automatic balance calculation
* Monthly payout reset
* Persistent history
* Offline-capable (IndexedDB)

## 🧱 Tech Stack

* React + Vite
* TypeScript
* Material UI
* React Router
* IndexedDB (idb)

## 🚀 Getting Started

```bash
npm install
npm run dev
```
