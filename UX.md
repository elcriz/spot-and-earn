# UX Rules

## Home Screen

* Animal buttons log sightings instantly
* Applies to all active children
* No confirmation dialogs

## Snackbar Undo

* Appears after logging
* Shows amount and child names
* Lasts ~4 seconds
* Undo removes last sighting only

## Kids Selection

* Toggle chips
* Persist selection across sessions

## Balances

* Derived from unpaid sightings
* "Mark all as paid" resets balances

## History

* Immutable log
* Reverse chronological

## Settings

* Add/remove/rename children
* Removing child does NOT break history

## Edge Cases

* No active kids → disable actions
* Undo only affects last entry
* Multiple rapid taps allowed
